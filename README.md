# ⌨️ BS Typing Game

**BS Typing Game** is a fun and interactive web-based typing practice application designed to improve typing speed, accuracy, and focus. It challenges users with randomly generated passages and provides real-time feedback on performance.

check the live app game: http://glistening-narwhal-5ed98a.netlify.app

Built using **HTML, CSS, and JavaScript**, the game is lightweight, fast, and runs directly in the browser without any installation.

---

## 🚀 Features

- 📝 Random typing passages each round  
- ⚡ Real-time WPM (Words Per Minute) tracking  
- 🎯 Accuracy calculation based on typed input  
- 📊 Progress bar to track completion  
- 🏆 Score system to measure performance  
- 🔄 Instant new passage after completion  
- 🎵 Background music support (optional)  
- 📱 Responsive and beginner-friendly UI  

---

## 🎮 How to Play

1. Open the game in your browser  
2. Start typing the displayed passage  
3. The system tracks your speed, accuracy, and score in real time  
4. Complete the passage to get a new challenge
